# Arquivos Jedi

Um site que reúne informações e links úteis em um só lugar, com foco em facilitar o acesso a recursos de estudo.

Acesse: https://o-tavio.github.io/arquivosjedi/

---

# Sobre

O Arquivos Jedi é um projeto que organiza conteúdos e materiais relevantes de forma simples e direta.

A proposta é centralizar links confiáveis e úteis, permitindo que qualquer pessoa encontre recursos de aprendizado com mais facilidade.

---

# Objetivo

Este projeto tem como objetivo:

- Reunir informações e links úteis em um único lugar  
- Organizar recursos para estudo  
- Facilitar o acesso a ferramentas e conteúdos educacionais  
- Apoiar a democratização do acesso à educação no Brasil  

---

# Estrutura
O projeto segue uma estrutura básica de site estático.
